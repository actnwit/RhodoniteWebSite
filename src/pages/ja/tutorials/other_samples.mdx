チュートリアルは以上ですが、もっと参考となるコードが欲しい場合は、Rhodoniteプロジェクトの[samples](https://github.com/actnwit/RhodoniteTS/tree/main/samples)ディレクトリを覗いてください。様々なサンプルのソースコードがあるのでコーディングの際の参考となるでしょう。

Rhodoniteプロジェクトをcloneし、プロジェクト直下で以下のコマンドを打つと、ローカルサーバーが起動します。

```
$ yarn start
yarn run v1.22.19
node server.js
Express server is running at http://localhost:8082
```

`http://localhost:8082`にアクセスすると、実際のサンプルの動作を確認することができます。
