## インストール

Rhodoniteを始めるには、以下の簡単な手順に従ってください：

<div className="steps">
  <div className="step">
    ### 1. リポジトリのクローン
    ```bash
    git clone https://github.com/actnwit/RhodoniteWebSite.git
    ```
  </div>

  <div className="step">
    ### 2. 依存関係のインストール
    ```bash
    npm install
    ```
  </div>

  <div className="step">
    ### 3. 開発サーバーの起動
    ```bash
    npm run dev
    ```
  </div>
</div>
