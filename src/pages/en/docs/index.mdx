# Getting Started with Rhodonite

<p className="lead">
  Welcome to Rhodonite! This guide will help you get started with our platform.

## Installation

To get started with Rhodonite, you'll need to follow these simple steps:

<div className="steps">
  <div className="step">
    ### 1. Clone the Repository
    ```bash
    git clone https://github.com/actnwit/RhodoniteWebSite.git
    ```
  </div>

  <div className="step">
    ### 2. Install Dependencies
    ```bash
    npm install
    ```
  </div>

  <div className="step">
    ### 3. Start the Development Server
    ```bash
    npm run dev
    ```
  </div>
</div>
